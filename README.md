# Veneury-SuperMarkeLasTresB

Este proyecto es un ejemplo de una página web sencilla para un supermercado ficticio llamado **SuperMarkeLasTresB**, desarrollada con HTML y CSS puro.

## 🛒 Características

- Página de inicio con bienvenida y carrusel de imágenes automático.
- Sección de productos destacados con tarjetas interactivas.
- Página de categorías que muestra productos al pasar el mouse.
- Página de contacto con formulario estilizado.
- Diseño responsive para dispositivos móviles.

## 📁 Estructura del proyecto

├── index.html # Página principal con carrusel y productos en oferta
├── productos.html # Lista de productos destacados
├── categorias.html # Categorías con interacción al pasar el mouse
├── contacto.html # Formulario de contacto estilizado
├── styles.css # Estilos generales del sitio
└── README.md # Este archivo

## 🛠 Tecnologías usadas

- HTML5
- CSS3 (sin frameworks, todo personalizado)

## 💡 Objetivo

El objetivo de este proyecto es demostrar cómo crear una interfaz simple, elegante y funcional utilizando solamente HTML y CSS, sin JavaScript ni librerías externas.

## 🔧 Cómo usar

1. Clona o descarga este repositorio.
2. Abre `index.html` en tu navegador.
3. Navega entre las páginas desde el menú superior.
